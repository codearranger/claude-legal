# or-pro-se — Tone and protocol

## Prompt

I'm representing myself in an Oregon debt-collection case. I
keep wanting to call the debt buyer's lawyer a liar in my
motion. Should I?

## Expected triggers

- `or-pro-se`

## Acceptance criteria

- [ ] Recommends AGAINST personal attacks; cites Parker
      framework principles (write to the judge, not to
      opposing counsel)
- [ ] Suggests neutral fact-based language ("Counsel's
      characterization differs from the record at Decl. ¶ 4")
- [ ] Explains why personal attacks hurt credibility with
      the judge
- [ ] References the OEC 803(6) record analysis as a
      stronger ground (let the record speak)
- [ ] Notes that pro se status doesn't change tone
      expectations

### Oregon-specific notes

- [ ] Uses "v." not "vs."
- [ ] References ORCP / UTCR if applicable
- [ ] Does NOT use "RCW" or "GR 14" (those are Washington)

## Common failure modes

- Validating the user's instinct to attack opposing counsel
- Failing to redirect to record-based critique
- Generic legal-writing advice without Oregon flavor
