# or-consumer-debt — Fees recovery for prevailing defendant

## Prompt

If I win my Oregon debt-defense case, what fees can I
recover? I'm pro se now but might hire counsel.

## Expected triggers

- `or-consumer-debt`
- `or-law-references` for fees-and-costs

## Acceptance criteria

### Identifies multiple fee paths

- [ ] **FDCPA § 1692k(a)(3)** — mandatory consumer fees
- [ ] **Oregon UTPA — ORS 646.638(3)** — mandatory
      prevailing-party fees
- [ ] **ORS 697.085** — Collection-agency act fees
- [ ] **ORS 20.105** — objectively unreasonable position
- [ ] **ORS 20.096** — reciprocal contract fees
- [ ] **ORS 20.190** — statutory prevailing-party fee
      ($200–$590)

### Pro se limitations

- [ ] Notes that pro se litigants **cannot recover attorney
      fees for their own time** (American Rule)
- [ ] Identifies what pro se CAN recover:
  - Costs (filing fees, postage, copying)
  - Statutory minimums under UTPA ($200), ORS 697 ($200)
  - ORS 20.190 prevailing-party fee ($200–$590)
  - Fees for any counsel retained later

### ORCP 68 procedure

- [ ] **14-day deadline** to file Statement of Attorney
      Fees under ORCP 68 C(2)
- [ ] Calendars this immediately upon judgment
- [ ] Notes the lodestar method (*Strawn*)
- [ ] References ORS 20.075 factors

### Strategic implications

- [ ] Retaining counsel may produce better outcome via fee
      recovery (especially if case is large)
- [ ] Calculating the cost-benefit

## Common failure modes

- Suggesting pro se can recover attorney-rate fees for own
  time
- Missing the 14-day ORCP 68 deadline
- Listing only FDCPA fees without UTPA / ORS 697 / ORS
  20.105 / ORS 20.190
- Generic fee advice without ORCP 68 procedure
