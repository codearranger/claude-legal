# or-consumer-debt — ORS 697.105 lack-of-capacity defense

## Prompt

I'm being sued by Velocity Investments, LLC, a Delaware
debt buyer, in Multnomah. They claim they bought my old
credit-card account. Do I have any Oregon-specific defenses
beyond the standard FDCPA stuff?

## Expected triggers

- `or-consumer-debt`

## Acceptance criteria

### Identifies ORS 697.105 lack-of-capacity defense

- [ ] Cites **ORS 697.005** definitions ("collection
      agency" includes debt buyers)
- [ ] Cites **ORS 697.015** registration requirement
- [ ] Cites **ORS 697.105** — unregistered collection
      agency cannot maintain an action

### Discovery / verification

- [ ] Directs user to DCBS database
      (https://orcfr.dcbs.oregon.gov/) to check
      registration status
- [ ] Recommends screenshot for the record

### Procedural

- [ ] Recommends pleading as **affirmative defense** in
      the answer
- [ ] Recommends ORCP 21 A(1) motion to dismiss for lack
      of capacity if registration is clearly missing
- [ ] Cites ORCP 47 summary judgment as an option once
      discovery confirms

### Counterclaim

- [ ] **ORS 697.085** — private right of action; $200
      statutory minimum + actual damages + fees
- [ ] **Oregon UTPA** — unregistered collection is "unfair
      conduct" under ORS 646.608(1)(u)

### Strategic value

- [ ] Explains that this is often a DISPOSITIVE defense —
      unregistered debt buyers cannot win the underlying
      suit
- [ ] Notes that re-registering mid-suit does NOT cure the
      defect

### What it should NOT do

- [ ] Should NOT generic "consult an attorney" without
      explaining the rule
- [ ] Should NOT recommend pleading FDCPA without UTPA
      (UTPA is broader for unregistered conduct)
- [ ] Should NOT skip the DCBS lookup

## Common failure modes

- Failing to identify the ORS 697.105 capacity defense
- Treating ORS 697 as just a "fines and penalties" statute
  rather than a defense
- Missing the strategic priority of this defense
- Generic FDCPA advice without the Oregon-specific layer
