# Integration — End-to-end motion to compel

## Prompt

I served First RFPs on Plaintiff debt buyer on April 1, 2025
and they served bogus objections on May 1, 2025. Meet-and-
confer happened May 10-14 without resolution. I want to file
a Motion to Compel in Multnomah Co. The case is 25CV01234.
Help me prepare the full packet.

## Expected triggers

Many skills should chain together:

- `or-discovery` (motion-to-compel framework)
- `or-draft-motion` (memorandum structure)
- `or-draft-declaration` (supporting declaration)
- `or-draft-order` (proposed order)
- `or-draft-note` (notice of hearing)
- `or-schedule-hearing` (JA contact for date)
- `or-file-packet` (packet assembly)
- `or-statewide-format` (UTCR 2.010 baseline)
- `or-multcc` (Multnomah specifics)
- `or-pro-se` (Parker framework)
- `or-quality-check` (final QC)
- `or-fact-check` (citation verification)

## Acceptance criteria

### Packet components produced

- [ ] Motion + Memorandum (Parker framework I–VII)
- [ ] Declaration of Defendant
- [ ] Exhibits (numbered)
- [ ] [PROPOSED] Order
- [ ] Notice of Hearing
- [ ] Certificate of Service

### Multnomah-specific compliance

- [ ] Meet-and-confer certification (Multnomah SLR 5.045)
- [ ] Working-copy threshold reminder (25 pages — Multnomah
      SLR 5.100)
- [ ] JA contact protocol for date reservation
- [ ] Notice of Hearing filed within 3 business days of JA
      confirmation (SLR 5.015)

### Substantive correctness

- [ ] Cites ORCP 46 A as the procedural authority
- [ ] Cites ORCP 46 A(4)(a) for fees
- [ ] Cites ORCP 36 B(1) for scope
- [ ] Cites OEC 803(6) / 902(11) if discussing authentication
- [ ] No Bluebook periods in citations
- [ ] Pro se signature blocks (no OSB#)
- [ ] Numbered exhibits

### Process

- [ ] Identifies the sequence: reserve date → draft → QC →
      fact-check → eFile → working copies if over threshold
- [ ] Recommends `or-fact-check` and `or-quality-check`
      before filing
- [ ] Sets reminders for hearing date and prep

## Common failure modes

- Missing one packet component
- Using "Note for Motion Docket" instead of "Notice of
  Hearing"
- Skipping the meet-and-confer certification
- Forgetting working copies for Multnomah
- Conflating Multnomah's JA-scheduling with Washington Co's
  Civil Division
- Citing CR 37 instead of ORCP 46
- Generic packet checklist without Multnomah specifics
