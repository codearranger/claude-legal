# or-hearings — WebEx hearing prep

## Prompt

My motion to compel hearing is on WebEx in Multnomah on July
15 at 10 AM. I'm pro se. What do I need to do to prepare?

## Expected triggers

- `or-hearings`
- Likely `or-multcc` for venue-specific notes

## Acceptance criteria

### Technical prep

- [ ] Recommends installing/testing WebEx 24-48 hours in
      advance
- [ ] Testing camera, microphone, speakers
- [ ] Logging in 10-15 minutes early
- [ ] Having a phone backup for call-in
- [ ] Ethernet vs Wi-Fi note
- [ ] Headphones recommendation

### Etiquette

- [ ] "Your Honor" form of address
- [ ] Stand when addressing the court (or as the Court
      directs)
- [ ] Mute when not speaking
- [ ] Camera at eye level
- [ ] Plain background; good lighting

### Argument prep

- [ ] References the pro-se drafting framework — 2-3 main points
- [ ] Anticipate judge questions
- [ ] Concede appropriate points
- [ ] Have outline and key exhibits printed
- [ ] Time limit awareness (10-20 minutes per side typical)

### Multnomah-specific

- [ ] Connection info from JA's confirmation email
- [ ] Each Multnomah judge's standing order may have
      specifics
- [ ] In-person fallback at Central Courthouse, 1200 SW 1st
      Ave

## Common failure modes

- Generic Zoom advice without WebEx specifics
- Suggesting "in-person" when Multnomah default is WebEx for
  routine motions
- Forgetting the call-in backup
- Missing the etiquette / formality reminder
