# co-quality-check — Pre-filing format + content QC (CJD 11-01 + Parker framework)

## Prompt

I've drafted a motion for summary judgment in Colorado and
I want to make sure it's ready before I file it. Walk me
through the pre-filing checklist — formatting, page limits,
citation accuracy, and anything else that could get it
bounced or embarrass me.

## Expected triggers

- `co-quality-check`
- `co-statewide-format`
- `co-fact-check`

## Acceptance criteria

### Format pass (CJD 11-01 + C.R.C.P. 10)

- [ ] Checks the **two-block Colorado caption** (left case
      block + COURT USE ONLY box), document title below
      the caption, line numbering, 1-inch margins, 12-point
      font, double spacing — reads the current standards
      from the references corpus rather than asserting fixed
      numbers from memory
- [ ] Verifies the **Certificate of Conferral** (C.R.C.P.
      121 § 1-15(8)) is present for non-dispositive motions
      and **absent** (or appropriately waived) for
      dispositive motions where conferral is not required

### Page limits (C.R.C.P. 121 § 1-15(1))

- [ ] Confirms the motion stays within the **C.R.C.P.
      121 § 1-15** page limits — reads the current
      15/15/10-page (motion/response/reply) limits from the
      skill references / corpus rather than asserting from
      memory; flags if the draft exceeds the limit

### Content pass (Parker framework)

- [ ] Runs the **Parker four-part check**: (1) relief
      requested stated clearly up front; (2) controlling
      rule or statute cited with the standard; (3) facts
      tied to the record with exhibit references; (4)
      conclusion restating requested relief
- [ ] Verifies every **citation** — rule numbers, statute
      sections, and case citations — against the references
      corpus or a structured source (no memory-asserted
      cites)

### Proposed order

- [ ] Confirms a **separate proposed order** is attached
      (cross-reference `co-draft-order`); not embedded in
      the motion body

## Common failure modes

- Accepts the page count without checking against the
  C.R.C.P. 121 limits
- Misses the Certificate of Conferral requirement (or
  includes one on a dispositive motion where it is not
  required)
- Lets an unverified case citation pass (asserts from
  memory instead of checking against the corpus)
- Misses a missing proposed order
