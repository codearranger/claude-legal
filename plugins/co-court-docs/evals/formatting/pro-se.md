# co-pro-se — Self-represented party workflows, JDF forms, and fee waivers

## Prompt

I can't afford a lawyer and I want to represent myself in
Colorado District Court. How do I file documents, do I need
to use official forms, and what if I can't afford the
filing fee?

## Expected triggers

- `co-pro-se`
- `co-statewide-format`
- `co-file-packet`

## Acceptance criteria

### Self-represented designation and signature

- [ ] Explains the **"Self-Represented"** designation on the
      signature block (no Attorney Reg. #) and that the
      litigant signs in their own name as "Pro Se" /
      "Self-Represented Party"
- [ ] Notes that the **same C.R.C.P. 10 + CJD 11-01 format
      rules** apply to self-represented filings as to
      attorney-filed documents; no special form exemption

### JDF forms catalog

- [ ] Identifies the **JDF (Judicial Department Forms)**
      catalog as the primary forms source for self-represented
      litigants (dissolution, child support, protection
      orders, etc.) and notes that using the official JDF
      form is mandatory where one exists for the relief
      sought — read the catalog scope from the skill
      references rather than asserting which specific JDF
      numbers are mandatory from memory
- [ ] Notes the **CCEFS Pro Se** e-filing track and that
      self-represented parties may also file by paper in
      many venues (cross-reference `co-file-packet`)

### Fee waivers

- [ ] Identifies **JDF 205 (Motion to File Without Payment of
      Fees)** and **JDF 206 (Order re: Motion to File
      Without Payment of Fees)** as the Colorado fee-waiver
      forms — reads the current income-eligibility standard
      from the references corpus rather than asserting
      specific dollar thresholds from memory
- [ ] Notes the court may set a hearing on the JDF 205
      before ruling on the waiver

### Self-Help resources

- [ ] Routes the self-represented litigant to the
      **Self-Help Center** directory and the courthouse
      resource desk — reading the per-judicial-district
      locations from the skill references rather than
      inventing specific room numbers or web addresses

## Common failure modes

- Adds an Attorney Reg. # to a self-represented signature
  block
- Tells the filer a different (relaxed) format standard
  applies to pro se documents
- Asserts the fee-waiver income threshold as a fixed dollar
  amount from memory
- Ignores JDF forms where one is officially mandated
