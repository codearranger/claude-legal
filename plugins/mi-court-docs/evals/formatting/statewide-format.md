# mi-statewide-format — MCR 1.109 / MCR 2.113 document format

## Prompt

I'm about to file my first document in a Michigan court and I
want to make sure it actually looks the way Michigan requires —
the caption, the heading, the signature, page setup. Can you
lay out the formatting rules so the clerk doesn't reject it?

## Expected triggers

- `mi-statewide-format`
- `mi-pro-se`

## Acceptance criteria

### Caption and heading (MCR 1.109 / MCR 2.113)

- [ ] States that the form and captioning of documents is
      governed by **MCR 1.109** (form of documents filed) and
      **MCR 2.113** (form of pleadings)
- [ ] Caption shows: the court (including the judicial circuit
      / district), the parties with their designations, the
      case number, and the assigned judge where required —
      cite the rule; read the current required caption elements
      from the references corpus rather than listing from memory
- [ ] Each document carries a **title** identifying its nature

### Page setup and legibility

- [ ] Describes the legibility / paper / margin / font
      requirements from **MCR 1.109** — cite the rule and **read
      the current specifics (margins, type size, paper size)
      from the references corpus** rather than asserting fixed
      numbers from memory
- [ ] Notes any line-numbering / footer expectations as governed
      by the rule and local practice (read current requirements
      from the corpus; do not invent a statewide mandate that
      isn't in the rule)

### Signature (MCR 1.109(E))

- [ ] States that every document must be **signed** under
      **MCR 1.109(E)**, with the signer's name, address, and
      telephone number; an attorney adds the **P-number**, a
      self-represented party signs without one
- [ ] Notes the MCR 1.109(E) signature certification (the
      signature certifies the document is well grounded, etc.)

## Common failure modes

- Cites federal or another state's format rules
- Asserts hard margin/font numbers as fixed facts instead of
  reading them from the corpus
- Omits the MCR 1.109(E) signature certification
- Adds a P-number to a self-represented signature block
