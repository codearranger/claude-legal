# ca-law-references — Evidence rules (business records)

## Prompt

The plaintiff in my debt-buyer case wants to introduce account
records under a custodian-of-records declaration. What's the
California rule?

## Expected triggers

- `ca-law-references`
- `ca-consumer-debt`

## Acceptance criteria

### Operative rules

- [ ] Cal. Evid. Code § 1271 — business-records exception to
      hearsay
- [ ] Cal. Evid. Code § 1561 — custodian-of-records affidavit
      (or declaration)
- [ ] Cal. Evid. Code § 1421 — handwriting authentication
- [ ] Cal. Evid. Code §§ 1500-1567 — Best Evidence / Secondary
      Evidence Rule (CA uses "Secondary Evidence Rule" under
      § 1521)

### Foundation elements (§ 1271)

- [ ] Made in the regular course of business
- [ ] At or near the time of the event
- [ ] By person with knowledge (or transmitted by such)
- [ ] Trustworthy

### CA vs. FRE distinction

- [ ] FRE 803(6) and 902(11) are analogous but distinct
- [ ] Differences in authentication: § 1561 declaration
      requirements
- [ ] California treats hearsay exceptions as enumerated (not
      a general "residual" exception)

### Debt-buyer application

- [ ] FDBPA (Cal. Civ. Code § 1788.58) imposes additional
      requirements for debt-buyer suits on credit card debt
- [ ] Chain-of-custody / chain-of-title attacks: original
      creditor records may not satisfy § 1271 foundation when
      submitted by debt buyer who never had business
      relationship with consumer

## Common failure modes

- Citing FRE 803(6) (federal) instead of Cal. Evid. Code § 1271
- Importing OR's OEC 803(6) language
- Confusing Best Evidence (federal FRE 1002) with CA's
  Secondary Evidence Rule (§ 1521)
- Missing the custodian-of-records declaration mechanism
  (§ 1561)
- Missing the FDBPA overlay for debt-buyer cases
