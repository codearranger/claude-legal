# ca-file-packet — Pre-filing packet preflight

## Prompt

I'm ready to file my motion to compel further responses in LASC.
What documents go in the packet and how do I e-file?

## Expected triggers

- `ca-file-packet`
- `ca-lasc`
- `ca-statewide-format`

## Acceptance criteria

### Packet components

- [ ] Notice of Motion (separate document)
- [ ] Memorandum of P&A (max 15 pages — CRC 3.1113(d); 20 with
      leave)
- [ ] Separate Statement — CRC 3.1345 (motion to compel further)
- [ ] Declaration(s) — including meet-and-confer declaration
- [ ] Proposed Order (separate document)
- [ ] Proof of Service

### Page limits

- [ ] Memo max 15 pages under CRC 3.1113(d)
- [ ] Table of contents + table of authorities required if
      over 10 pages — CRC 3.1113(f)

### E-filing (LASC)

- [ ] Mandatory for represented parties (CRC 2.250+)
- [ ] LASC uses Odyssey eFileCA (Tyler-based) via lacourt.org
- [ ] PDF with bookmarks for exhibits ≥ 25 pages (CRC
      3.1110(f)(4))

### Filing fee

- [ ] Unlimited civil first appearance: ~$435 (Govt. Code
      § 70611)
- [ ] Motion fee: ~$60 (Govt. Code § 70617)
- [ ] Fee waiver: Forms FW-001 / FW-003 (Govt. Code §§ 68630-
      68641)

### Service / timing

- [ ] Notice service 16 court days before hearing (CCP § 1005(b))
- [ ] Adjust for mail (5 cal days) or e-service (2 court days)

## Common failure modes

- Combining Notice + Memo into one document
- Missing the separate statement (CRC 3.1345)
- Exceeding the 15-page memo limit without leave
- Failing to bookmark large exhibit PDFs
- Importing OR's File and Serve or WA's KCDC/KCSC portal
- Wrong filing-fee amount (refer to Civil Filing Fee schedule
  on lacourt.org)
