# ca-hearings — Tentative-ruling regime

## Prompt

I have a hearing on my motion to compel in LASC tomorrow morning.
What should I expect?

## Expected triggers

- `ca-hearings`
- `ca-lasc`

## Acceptance criteria

### Tentative-ruling protocol

- [ ] CRC 3.1308(a)(1) — tentative posted by court
- [ ] LASC Local Rule 3.31 — tentative posted day before
      (typically by 4:30 p.m.)
- [ ] If no party contests by 4:30 p.m. day before, tentative
      becomes order — no hearing
- [ ] To contest: call other party + department by 4:30 p.m.
      day before
- [ ] Failing to contest = tentative becomes the order

### Hearing format

- [ ] Oral argument typically 5-10 min per side
- [ ] In-person or remote (LACourtConnect for LASC)
- [ ] Address judge as "Your Honor"
- [ ] State appearance for the record at start

### Remote appearances

- [ ] CRC 3.672 — generally allowed post-COVID
- [ ] LASC uses LACourtConnect (lacourtconnect.org)
- [ ] Schedule remote appearance in advance per dept. rules

### What to bring

- [ ] Copy of motion + opposition + reply
- [ ] Tentative ruling printout
- [ ] Argument outline
- [ ] Proposed order (if asked to submit)

## Common failure modes

- Missing the tentative-ruling system entirely (CA-specific
  procedural feature; not used in OR or WA)
- Failing to call to contest by 4:30 p.m. = inadvertent waiver
- Importing OR's "Notice of Hearing" date as the only date
- Importing Multnomah's contact-the-JA protocol
- Failing to specify LACourtConnect for LASC remote
  appearances
