# ca-pro-se — Pro per tone and protocol

## Prompt

I'm representing myself in a debt collection case in California.
How should my filings differ from what an attorney would do?

## Expected triggers

- `ca-pro-se`
- `ca-statewide-format`

## Acceptance criteria

### Terminology

- [ ] "In Pro Per" or "Self-Represented Litigant" — California
      uses both
- [ ] NOT "Pro se" (less common in CA filings; Oregon usage)
- [ ] NO bar number in signature block

### Caption / signature

- [ ] Attorney info block in caption shows name + address +
      phone + email + "In Pro Per"
- [ ] No "Esq." after name

### pro-se drafting framework adapted

- [ ] Short introduction (3-5 sentences)
- [ ] Lead with strongest legal point
- [ ] Record citations to declaration paragraphs / exhibits
- [ ] Avoid emotional / personal-grievance language

### Resources mentioned

- [ ] Self-Help Centers at every Superior Court
- [ ] courts.ca.gov/selfhelp
- [ ] lawhelpca.org
- [ ] Fee waiver (Forms FW-001 / FW-003)

### Service conventions

- [ ] Proof of Service (Forms POS-040 / POS-030)
- [ ] E-service to pro per requires consent (CCP § 1010.6)
- [ ] Mail service adds 5 calendar days (CCP § 1013(a))

## Common failure modes

- Using "Pro Se" instead of "In Pro Per" (CA convention)
- Including a bar number / "Esq." for an unrepresented party
- Importing OR's "self-represented litigant" only — CA accepts
  both terms; "In Pro Per" is more common
- Citing Washington pro-se drafting framework cases verbatim
- Failing to note fee-waiver forms
