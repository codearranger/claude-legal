# ca-statewide-format — CRC 2.100-2.119 compliance

## Prompt

I'm drafting a motion to file in California Superior Court. What
are the formatting requirements I need to follow?

## Expected triggers

- `ca-statewide-format`

## Acceptance criteria

### Paper / margins

- [ ] Letter (8.5 × 11) — CRC 2.104
- [ ] White, opaque, unglazed paper — CRC 2.103
- [ ] One side only — CRC 2.103
- [ ] 1-inch margins all sides — CRC 2.107
- [ ] Two-hole punch at top — CRC 2.119

### Type

- [ ] 12-point minimum — CRC 2.104
- [ ] "Essentially equivalent to Courier, Times New Roman, or
      Arial" — CRC 2.105
- [ ] Roman style (not italic body)

### Line spacing / numbering

- [ ] 1.5 or double-spaced — CRC 2.108(a)
- [ ] Lines numbered in the left margin (typically 1-28 per
      page) — CRC 2.108(b)
- [ ] Page numbers in footer (CRC 2.109)

### Ink / color

- [ ] Black ink only on white paper — CRC 2.103 (stricter than
      Oregon UTCR 2.010 "black or blue-black")

### Footer / pagination

- [ ] Footer with title of paper + page number (CRC 2.110)
- [ ] "Page X of Y" recommended

### Caption (CRC 2.111)

- [ ] Attorney info upper-left within 2.5 inches of left edge
- [ ] Case caption block on right (court, county, case number,
      title, hearing info)

### Citation format (California Style Manual)

- [ ] "Code Civ. Proc., § 437c" (with comma + § symbol)
- [ ] "Cal. Rules of Court, rule 3.1308" (not "CRC 3.1308" in
      formal citations)
- [ ] "Cal." reporter (not "Wn." or "Or.")

## Common failure modes

- Importing OR's UTCR 2.010 "black or blue-black" — CRC 2.103
  is stricter (black only)
- Importing WA's GR 14 margin/font conventions — CA differs
- Failing to mention line numbers (CRC 2.108(b))
- Forgetting two-hole punch (CRC 2.119) — California-specific
- Citing rules as "CRC X" rather than "Cal. Rules of Court,
  rule X" in formal citations
