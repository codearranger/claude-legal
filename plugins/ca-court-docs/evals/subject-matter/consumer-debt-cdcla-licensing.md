# ca-consumer-debt — CDCLA licensing defense

## Prompt

A debt collector is suing me in California. Do they need a license
to collect debts in this state?

## Expected triggers

- `ca-consumer-debt`

## Acceptance criteria

### CDCLA overview

- [ ] **California Debt Collection Licensing Act** (Cal. Fin.
      Code §§ 100000-100027)
- [ ] **Effective January 1, 2022** — both debt collectors AND
      debt buyers must be licensed by DFPI
- [ ] Pre-2022 there was no CA licensure regime for debt
      collectors (unlike OR's ORS 697 since 1989)

### Application to suit

- [ ] Non-licensure is an **affirmative defense**
- [ ] Suing without license = potential violation
- [ ] Predicate for UCL "unlawful" prong (Bus. & Prof. Code
      § 17200)

### License verification

- [ ] DFPI license search at dfpi.ca.gov — public database
- [ ] Verify before answer / cross-complaint

### Discovery hooks

- [ ] RFP for CDCLA license certificate
- [ ] RFA admissions on license status

### Rosenthal + CDCLA overlap

- [ ] CDCLA references Rosenthal Act (Fin. Code § 100002(d))
- [ ] Cumulative claims possible

### Exemptions

- [ ] Some collectors exempt (e.g., licensed attorneys
      collecting in litigation — Fin. Code § 100001(e))
- [ ] Verify whether plaintiff fits an exemption

## Common failure modes

- Reporting CDCLA effective date as pre-2022
- Importing Oregon's ORS 697 (Collection Agency Registration)
  framework
- Missing the DFPI license search step
- Failing to plead non-licensure as affirmative defense
- Failing to use CDCLA as UCL predicate
- Treating CDCLA as a counterclaim cause of action (it's
  primarily an affirmative defense + UCL predicate; statute
  may not create a stand-alone private right of action)
