# California County Courts Directory

A reference directory for the nine most-populous California
counties (after Los Angeles and San Francisco). For Los Angeles
use `ca-lasc`; for San Francisco use `ca-sfsc`.

Pull current contacts from each court's website — addresses and
phone numbers change, and clerk offices may move within a building.

---

### eFiling

- **Platform**: Tyler Technologies Odyssey eFileCA
- **Portal**: **https://sdcourt.ca.gov/efiling/**

### Case numbers

| Type | Format | Example |
|------|--------|---------|
| Unlimited civil (Central) | `37-YYYY-NNNNNNN-CU-[type]-CTL` | `37-2024-00012345-CU-BC-CTL` |

| Limited civil | `37-YYYY-NNNNNNN-CL-[type]` | `37-2024-00012345-CL-CL` |

### Tentative rulings

Posted by **1:30 p.m.** the court day before the hearing at
**https://www.sdcourt.ca.gov/tentativerulingsearch/**
Contest by calling the specific department by 4:00 p.m.

### Local rules

San Diego Superior Court Local Rules at **sdcourt.ca.gov/localrules/**

Key civil rules:
- SDSC Local Rule 2.1: eFiling requirements
- SDSC Local Rule 3.1: case management; CMC at 90-120 days
- SDSC Local Rule 5.1: discovery; meet-and-confer requirements
- SDSC Local Rule 8.1: tentative rulings
