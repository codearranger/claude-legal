# tn-statewide-format — Baseline Tennessee formatting (Rule 10 + local rules)

## Prompt

What are the formatting requirements for a motion I'm filing
in a Tennessee Circuit Court? Margins, font, caption — the
whole thing. I'm new to Tennessee practice.

## Expected triggers

- `tn-statewide-format`

## Acceptance criteria

### The architectural point (must get this right)

- [ ] Correctly states that **Tennessee has NO single
      statewide page/margin/font formatting rule** — typography
      and page limits are set by **per-county local rules**,
      not by a statewide rule
- [ ] Directs the litigant to confirm the controlling **local
      rules** for the venue (AOC "Local Rules of Practice"
      index at tncourts.gov) before relying on any default

### Common-practice defaults

- [ ] Gives sensible common-practice defaults while labeling
      them as defaults (not a statewide mandate): **Letter-size
      (8.5 x 11)** paper, **1-inch margins**, **12-pt** body
      font
- [ ] Notes line numbering and a footer/identification block
      as common conventions, subject to local rules

### What the statewide rules actually govern

- [ ] **Tenn. R. Civ. P. 10.01 caption**: court, parties,
      file/docket number, and pleading designation (per Rule
      7.01)
- [ ] Tenn. R. Civ. P. **10.02** numbered paragraphs and
      **10.03** attachment of written instruments
- [ ] Tenn. R. Civ. P. **11** signature requirement
- [ ] Redaction of personal identifiers (cite the controlling
      rule; read the current redaction-rule citation from the
      references corpus)

### Citation style

- [ ] Standard Bluebook; cases in South Western Reporter
      (S.W., S.W.2d, S.W.3d); statutes as Tenn. Code Ann. /
      T.C.A. § NN-N-NNN

## Common failure modes

- Invents a fictitious "statewide format rule" with hard
  margin/font numbers as a mandate
- Fails to defer typography and page limits to local rules
- Omits the Rule 10.01 caption elements
- Treats common-practice defaults as binding statewide law
