# oh-discovery — Motion-to-compel framework

## Prompt

I served interrogatories and document requests in my
Ohio Common Pleas case 40 days ago. The defendant has not
responded at all. What's my next step?

## Expected triggers

- `oh-discovery`
- `oh-draft-motion`

## Acceptance criteria

- [ ] Identifies **Civ. R. 33** (Interrogatories — 40-
      question cap) and **Civ. R. 34** (Requests for
      Production) as the governing rules
- [ ] Identifies the 28-day response deadline under each
- [ ] Walks through **Civ. R. 37(E)** good-faith
      conferral / meet-and-confer requirement
- [ ] Walks through **Civ. R. 37(A)** motion-to-compel
      framework
- [ ] References available sanctions: Civ. R. 37(A)(5)
      fees + costs; Civ. R. 37(B) failure-to-comply
      sanctions
- [ ] Does NOT default to FRCP language

## Common failure modes

- Skipping meet-and-confer
- Citing FRCP 37 only
- Wrong response deadline
- No reference to Civ. R. 33 40-question cap
