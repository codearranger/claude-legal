# oh-family-law — Divorce vs. dissolution

## Prompt

My spouse and I both want to end our marriage in Ohio.
We agree on everything (no kids, modest assets). What's
the fastest path?

## Expected triggers

- `oh-family-law`
- `oh-family-court`

## Acceptance criteria

- [ ] Identifies **dissolution** under **R.C. 3105.61-65**
      as the agreed-on uncontested path
- [ ] Walks the joint-petition + separation-agreement
      requirement
- [ ] Notes the **30-90 day window** to final hearing
      under R.C. 3105.64
- [ ] Contrasts with **divorce** under R.C. 3105.01 +
      grounds + residency (6 months state / 90 days county)
- [ ] References mandatory financial disclosures
- [ ] Notes no-fault "incompatibility" ground requires
      both parties to admit incompatibility
- [ ] Notes annulment under R.C. 3105.31 as separate
      narrow remedy

## Common failure modes

- Conflating divorce with dissolution
- Missing the residency requirement
- Missing the separation-agreement scope for dissolution
- Wrong waiting period
