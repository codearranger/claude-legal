# oh-cuya — Cuyahoga County Common Pleas conventions

## Prompt

I have a civil case in Cuyahoga County Common Pleas. What
do I need to know about local rules and e-filing?

## Expected triggers

- `oh-cuya`
- `oh-statewide-format`
- `oh-law-references`

## Acceptance criteria

### Court basics

- [ ] Names Justice Center (1200 Ontario Street, Cleveland)
- [ ] References Cuyahoga County 8th District Court of
      Appeals
- [ ] References Cuyahoga County e-filing portal
- [ ] References the General Division and any specialized
      dockets

### Local rules

- [ ] References Cuyahoga Common Pleas Local Rules
- [ ] References working-copy / chambers-copy
      conventions if any
- [ ] References certificate of service local form if any

## Common failure modes

- Conflating Cuyahoga COMMON PLEAS with Cleveland
  MUNICIPAL Court (different courts, different rules)
- Wrong e-filing portal
- Citing OR / WA / CA local rules by mistake
