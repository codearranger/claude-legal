# oh-statewide-format — Civ. R. 10 caption + signature

## Prompt

I'm drafting an Answer in an Ohio Common Pleas case. What
does the caption look like, and what goes in the signature
block? Pro se.

## Expected triggers

- `oh-statewide-format`
- `oh-pro-se`

## Acceptance criteria

### Caption

- [ ] Court name in full caps: `IN THE COURT OF COMMON
      PLEAS OF [COUNTY] COUNTY, OHIO`
- [ ] Division line if applicable (CIVIL DIVISION)
- [ ] Case number with Ohio prefix convention (CV / DR)
- [ ] Parties: `vs.` with periods (Ohio convention)
- [ ] Title in caps: e.g., `DEFENDANT'S ANSWER`

### Paper format

- [ ] Letter paper 8.5 x 11
- [ ] 1.5-inch top margin on first page (caption); 1-inch
      on subsequent
- [ ] 1-inch sides + bottom
- [ ] 12-point minimum font (Times New Roman preferred)
- [ ] Double-spaced body
- [ ] Page numbering in footer

### Signature block (pro se)

- [ ] Name + address + phone + email
- [ ] Designation `Pro Se` or `Self-Represented`
- [ ] **NO** attorney registration number (don't fabricate)
- [ ] Signature line

### Citation format

- [ ] References Ohio public-domain citation
      `YYYY-Ohio-NNNN` for appellate cases (mandatory
      since 2002)
- [ ] References `R.C.` (not "ORC" or "Ohio Rev. Code")
      for in-state practice

## Common failure modes

- Pro se filing with a fabricated Atty. Reg. #
- "v." instead of "vs."
- Wrong citation format (e.g., West reporter only without
  public-domain cite)
- Single-spaced body
