# oh-pro-se — Pro se drafting framework

## Prompt

I'm representing myself in an Ohio Common Pleas civil case.
The other side is represented. What should I know about
how my filings should look and read?

## Expected triggers

- `oh-pro-se`
- `oh-statewide-format`

## Acceptance criteria

### Designation

- [ ] Signature block uses `Pro Se` or `Self-Represented`
- [ ] **NO** Atty. Reg. # appears
- [ ] Address + phone + email included
- [ ] Identifies role (Plaintiff / Defendant) clearly

### Tone framework

- [ ] Plain English; no legalese affectation
- [ ] Avoids "Comes now the Plaintiff" archaisms unless
      the local culture expects it
- [ ] Cites Civ. R. and R.C. accurately (not made-up
      rules)
- [ ] States facts before argument; numbered paragraphs in
      pleadings; lettered subparts where appropriate

### Affidavit (not declaration)

- [ ] Pro se affidavits MUST be notarized — Ohio does NOT
      accept self-verified "declarations under penalty of
      perjury" in civil practice (R.C. 2319.04; Civ. R.
      56(E))

## Common failure modes

- Fabricating an Atty. Reg. #
- Filing a "declaration under penalty of perjury" without
  notarization
- Using federal forms / federal-court terminology
- Overly aggressive tone (poor strategic posture)
