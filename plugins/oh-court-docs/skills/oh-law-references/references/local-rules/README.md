# local-rules Corpus — Ohio (per-court verbatim)

Verbatim mirrors of individual Ohio Court of Common Pleas **local
rules**, for drafting reference only. This is a departure from the
plugin's default pointer-based posture for local rules — carried
verbatim for the highest-touch courts where the local rules
materially shape filing practice.

> **NOT LEGAL ADVICE.** Local rules change frequently. Always verify
> against the court's current published version before filing.

## How these were produced

The courts publish their local rules as consolidated PDFs. Each file
here was produced by downloading the source PDF, extracting the text,
and stripping repeated page header/footer noise. The body text is
verbatim (public-domain court rules). Re-mirror by re-downloading the
source PDF above and repeating the extraction; update the version +
date in the file's header block.
