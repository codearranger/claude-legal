# az-draft-note — Notice of hearing / setting under Ariz. R. Civ. P. 7.1

## Prompt

I'm self-represented in Arizona Superior Court and I filed a
motion. I need to set it for oral argument and tell the other side
when. How do I draft the notice, and do I have to request oral
argument in a particular way?

## Expected triggers

- `az-draft-note`
- `az-statewide-format`

## Acceptance criteria

### Document identification

- [ ] Identifies the filing under **Ariz. R. Civ. P. 7.1**
      (motion practice — request and setting of oral argument /
      hearing) and frames it as the notice/request that surfaces a
      pending motion for a ruling or argument
- [ ] Explains that a request for oral argument must be made in the
      manner Rule 7.1 prescribes — read the current mechanics
      (where the request goes, default timing) from the references
      corpus rather than asserting them from memory
- [ ] Notes that many motions are decided without oral argument
      unless argument is timely requested (read the rule's default
      from the corpus)

### Caption and structure

- [ ] Caption per **Ariz. R. Civ. P. 10** matching the underlying
      motion; title identifies it as a Notice of Hearing / Request
      for Oral Argument
- [ ] Identifies the motion to be heard and leaves date/time/
      division placeholders consistent with how the court issues or
      confirms the setting (read from the corpus / cross-reference
      `az-schedule-hearing`)

### Service and composition

- [ ] Includes a certificate of service per **Ariz. R. Civ. P. 5**
      (read service mechanics from the corpus)
- [ ] Self-represented signature block; no bar number

## Common failure modes

- Treats the notice as self-scheduling when the court controls the
  setting, or vice versa, without reading Rule 7.1's mechanics
- Asserts oral-argument-request timing from memory
- Omits the certificate of service
