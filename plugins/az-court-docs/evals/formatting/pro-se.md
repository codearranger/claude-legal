# az-pro-se — Self-represented signature (no bar number) + self-service forms

## Prompt

I can't afford a lawyer and I'm handling my Arizona case myself.
How do I sign my documents as a self-represented person, and are
there official Arizona forms or self-service resources I should be
using instead of drafting everything from scratch?

## Expected triggers

- `az-pro-se`
- `az-statewide-format`

## Acceptance criteria

### Self-represented designation

- [ ] States that a self-represented party signs **without a State
      Bar of Arizona number** and uses a "self-represented" /
      "in propria persona" designation in the signature block — read
      the exact block format from the corpus
- [ ] Ties the signature requirement to the Arizona signing rule
      (Ariz. R. Civ. P. 11 / Rule 10 signature) by number rather
      than asserting the requirement generically

### Self-service resources

- [ ] Points to Arizona's self-service resources — the Judicial
      Branch / Superior Court **self-service center** forms and any
      court-approved fillable forms — read the current pointers from
      the references corpus / `online-sources.md` rather than
      inventing URLs or form numbers
- [ ] Notes that court-approved forms, where they exist, are
      preferable to ad-hoc drafting for self-represented filers

### Honest sourcing

- [ ] Reads form names/URLs and the signature-block format from the
      corpus rather than asserting them from memory

## Common failure modes

- Adds a bar number or attorney designation to a pro-se signature
- Invents Arizona self-service form numbers or URLs
- Asserts the signature-block wording from memory instead of the
  corpus
