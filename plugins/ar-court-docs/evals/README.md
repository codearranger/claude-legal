# Evals — Skill Regression Tests (Arkansas)

Prompt-based regression tests for the `ar-court-docs` plugin — **24 evals**
across the five categories. Each eval states a realistic pro-se prompt, the
skills it should trigger, and thin-skill acceptance criteria (criteria require
reading drifting figures — dollar caps, day counts — from the references corpus
rather than hard-coding them).

## Folder layout

- `procedural/` (8) — civil + evidence references, deadlines (Daisy Gatson Bates
  Day / Christmas Eve holidays), discovery (interrogatories available under
  Rule 33), fact-check (medium-neutral citation), first-30-days (Arkansas
  fact-pleading + Rule 12(b)(6)), post-judgment (Rule 60(a) 90-day window),
  District Court de novo appeal.
- `drafting/` (4) — motion (the *Wallace v. Broyles* "meet proof with proof" SJ
  standard), declaration/affidavit, Notice of Hearing, proposed order.
- `formatting/` (4) — statewide-format (no statewide format rule; Admin. Orders
  19/21), Pulaski venue, pro se, quality-check (local-rule WARN posture).
- `subject-matter/` (6) — consumer-debt triage / SOL split / collection-agency
  licensing, family-law (Act 604 joint-custody presumption + covenant marriage),
  landlord-tenant (Act 1052 habitability + criminal failure-to-vacate),
  personal-injury (comparative fault + no damages caps).
- `integration/` (2) — end-to-end debt-defense answer; motion-to-compel packet.

## Arkansas-distinctive coverage

Amendment 80 unified Circuit Court; no statewide page/font rule; fact-pleading;
the Rule 60(a) 90-day window; medium-neutral citation (Ark. Sup. Ct. R. 5-2);
the ADTPA (Act 986 of 2017 narrowing) + collection-agency licensing; the 5-year
written / 3-year open-account SOL split; covenant marriage + the Act 604 (2021)
joint-custody presumption + income-shares child support; the Act 1052 (2021)
implied-habitability standard + criminal failure-to-vacate; modified comparative
fault with the "less than" bar + no damages caps + State Claims Commission
immunity; the District Court de novo appeal.
