# ar-draft-order — Proposed order and approval as to form

## Prompt

The judge ruled in my favor in Arkansas. Draft the order.

## Expected triggers

- `ar-draft-order`
- `ar-submit-order`

## Acceptance criteria

- [ ] Produces a proposed order with separated findings and decretal paragraphs
- [ ] Flags that **only the judge signs**; the filer prepares a proposed form
- [ ] Describes circulating the order to opposing counsel "**approved as to
      form**" before submission
- [ ] Notes entry-of-order mechanics and cross-references `ar-submit-order`

## Common failure modes

- Has the party "sign" the order
