# ar-draft-declaration — Affidavit vs. unsworn declaration

## Prompt

I need a sworn statement to support my Arkansas motion. How should it be set up?

## Expected triggers

- `ar-draft-declaration`

## Acceptance criteria

- [ ] Defaults to a **notarized affidavit** with an Arkansas jurat
      (STATE OF ARKANSAS / COUNTY OF ___ / "subscribed and sworn")
- [ ] Notes the personal-knowledge requirement for affidavits used with
      summary judgment
- [ ] Explains when an **unsworn declaration under penalty of perjury** is
      acceptable vs. when a notarized affidavit is required
- [ ] Keeps the affidavit on line-numbered pleading paper with the footer

## Common failure modes

- Uses a non-Arkansas declaration form as the default without flagging the
  affidavit requirement
