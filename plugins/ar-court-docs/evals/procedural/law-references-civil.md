# ar-law-references — Arkansas civil rules lookup

## Prompt

What are the key Arkansas Rules of Civil Procedure I should know for a civil case?

## Expected triggers

- `ar-law-references`

## Acceptance criteria

- [ ] Identifies the rule set as the **Arkansas Rules of Civil Procedure**
- [ ] Notes Arkansas is a **fact-pleading** jurisdiction (Rule 8 requires facts)
- [ ] Points to the **Rule 60(a) 90-day window** vs. the narrower Rule 60(c) grounds
- [ ] Reads exact thresholds/day counts from the references corpus rather than
      asserting fixed numbers from memory

## Common failure modes

- Describes Arkansas as a notice-pleading jurisdiction
