# ar-post-judgment — Rule 60(a) 90-day window

## Prompt

A default judgment was entered against me in Arkansas two months ago. Can the
court still set it aside?

## Expected triggers

- `ar-post-judgment`

## Acceptance criteria

- [ ] Distinguishes **Ark. R. Civ. P. 60(a)** — the court may modify or vacate to
      correct error **within 90 days** of filing the judgment — from the narrower
      **Rule 60(c)** grounds available after 90 days
- [ ] Notes the Rule 59 new-trial motion window (read the day count from references)
- [ ] Mentions the 10-year judgment life and scire facias revival

## Common failure modes

- Treats the 90-day Rule 60(a) window and the post-90-day Rule 60(c) grounds as
  interchangeable
