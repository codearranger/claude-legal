# ar-first-30-days — Fact-pleading and the Rule 12(b)(6) motion

## Prompt

I was just served with a complaint in Arkansas. The complaint seems vague. What
are my options in the first 30 days?

## Expected triggers

- `ar-first-30-days`

## Acceptance criteria

- [ ] Identifies the answer window under **Ark. R. Civ. P. 12(a)** (read the day
      count from references) and the Rule 12(b) motion options
- [ ] Flags that Arkansas is a **fact-pleading** jurisdiction — the complaint
      must state **facts**, not conclusions — and explains the **Rule 12(b)(6)**
      motion to dismiss for failure to state facts upon which relief can be granted
- [ ] Covers affirmative defenses (Rule 8) and counterclaims (Rule 13)

## Common failure modes

- Applies a federal notice-pleading plausibility analysis without the Arkansas
  fact-pleading standard
