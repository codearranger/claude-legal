# ar-statewide-format — No statewide format rule; Rule 10/11 + Admin Orders

## Prompt

What are the formatting rules for a pleading I'm filing in an Arkansas circuit court?

## Expected triggers

- `ar-statewide-format`

## Acceptance criteria

- [ ] States that Arkansas has **no single statewide page/margin/font rule**;
      form is governed by **Ark. R. Civ. P. 10** (caption + numbered paragraphs)
      and **Rule 11** (signature), with typography from local rules
- [ ] Mentions **Administrative Order No. 19** (redaction of confidential
      information + certificate of compliance) and **Administrative Order
      No. 21** (eFlex e-filing)
- [ ] Gives the Arkansas caption shape (`IN THE CIRCUIT COURT OF [COUNTY]
      COUNTY, ARKANSAS`, optional division line, `Case No.`)
- [ ] States the medium-neutral citation form (`2015 Ark. 100` /
      `2015 Ark. App. 200`) under **Ark. Sup. Ct. R. 5-2**
- [ ] Describes line-numbered pleading paper and the "Page X of Y" footer as
      marketplace defaults

## Common failure modes

- Asserts a fixed statewide margin/font mandate as if Arkansas had one
