# Example skeleton — Arkansas Complaint for Divorce (illustrative)

> **NOT LEGAL ADVICE.** Illustrative skeleton only — not a form to file
> as-is. Verify every ground, residency count, and citation against
> current Ark. Code Ann. Title 9, ch. 12 and the venue's local
> administrative plan. Use `ar-statewide-format` for the caption and
> the Administrative Order No. 19 redaction/certificate.

This skeleton shows where the Arkansas-distinctive elements (grounds,
**corroboration**, residency, the one-half property presumption, the
Act 604 joint-custody presumption) appear in a divorce complaint. It is
a drafting aid, not a substitute for the substantive references.

```
            IN THE CIRCUIT COURT OF PULASKI COUNTY, ARKANSAS
                    DOMESTIC RELATIONS DIVISION

JANE DOE,                                  )
        Plaintiff,                         )
                                           )
v.                                         )   Case No. ____________
                                           )
JOHN DOE,                                  )
        Defendant.                         )

                     COMPLAINT FOR DIVORCE

   Plaintiff, for her Complaint for Divorce against Defendant,
states:

1.  RESIDENCY. Plaintiff is a resident of Arkansas and has resided
    in this State for the period required by Ark. Code Ann.
    § 9-12-307. [Be prepared to CORROBORATE residence.]

2.  MARRIAGE. The parties were married on [date] and [are / are not]
    a covenant marriage under Ark. Code Ann. § 9-11-801 et seq.
    [If covenant marriage, plead a covenant ground — see
    covenant-marriage.md.]

3.  CHILDREN / UCCJEA. [No minor children] OR [There are __ minor
    children; Arkansas is the children's home state under the UCCJEA,
    Ark. Code Ann. § 9-19-101 et seq.; UCCJEA affidavit attached.]

4.  GROUNDS. [General indignities under Ark. Code Ann. § 9-12-301]
    OR [The parties have lived separate and apart without
    cohabitation for eighteen (18) continuous months,
    § 9-12-301(b).] [Be prepared to CORROBORATE the ground.]

5.  PROPERTY. The parties own marital property to be divided under
    Ark. Code Ann. § 9-12-315, with the statutory presumption of an
    equal (one-half) division; and non-marital property to be
    returned to its owner.

6.  CUSTODY (if children). Custody should be determined in the
    children's best interest under Ark. Code Ann. § 9-13-101,
    subject to the rebuttable presumption that joint custody is in
    the best interest of the children (Act 604 of 2021).

7.  CHILD SUPPORT (if children). Support should be set under the
    income-shares Family Support Chart (Administrative Order No. 10).

8.  ALIMONY (if sought). Plaintiff seeks spousal support under
    Ark. Code Ann. § 9-12-312.

   WHEREFORE, Plaintiff prays for an absolute divorce; division of
property under § 9-12-315; [custody, support, alimony]; and all
proper relief.

                              ________________________________
                              Jane Doe, Pro Se / Self-Represented
                              [address, phone, email]
```

Pair with: `dissolution.md` (grounds, corroboration, residency,
waiting period), `property-distribution.md`, `custody-and-parenting.md`,
`child-support.md`, `maintenance.md`, and `ar-family-court` for venue.
