# Immigration Statutes — Immigration and Nationality Act (INA)

The Immigration and Nationality Act of 1952, as amended, is the core federal
immigration statute. It is codified at **Title 8 of the U.S. Code, Chapter 12**
("Immigration and Nationality"), **8 U.S.C. §§ 1101 et seq.** Verbatim text,
one Markdown file per subchapter.

§ 212(a)(6)(C)(i)"), while the codified text in these files is numbered by
**8 U.S.C. section**. The offset is *not* constant across the Act, so use this
crosswalk for the most-cited provisions. (The OLRC publishes the full table at
