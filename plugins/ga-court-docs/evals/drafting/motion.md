# ga-draft-motion — O.C.G.A. § 9-11-7 motion with USCR 6 briefing + Rule Nisi

## Prompt

The debt buyer suing me in the State Court of Cobb County ignored my
interrogatories and requests for production — boilerplate objections and
no documents. I want to file a motion to compel. Draft it for me the
right way, and tell me how a hearing gets set in Georgia.

## Expected triggers

- `ga-draft-motion`
- `ga-discovery`
- `ga-statewide-format`

## Acceptance criteria

### Motion structure and authority

- [ ] Builds the motion to compel under **O.C.G.A. § 9-11-37(a)** with
      the specific deficiency identified per request and the requests and
      responses attached as exhibits — reads the current rule text from
      the references corpus rather than asserting elements from memory
- [ ] Frames the motion as a pleading/motion under **O.C.G.A. § 9-11-7**
      and signs it per **O.C.G.A. § 9-11-11** (party signature; no general
      verification requirement)
- [ ] Notes the **USCR 6.2** thirty-day response window for the opposing
      party — reads the day count from the corpus rather than asserting it
      from memory

### Hearing and Rule Nisi

- [ ] Explains the Georgia **Rule Nisi** practice (**USCR 6.3 / 6.4**):
      oral argument on a motion is obtained by submitting a proposed Rule
      Nisi for the judge to set a hearing date — the party does not
      self-calendar — reads the per-court hearing-request practice from
      the skill references
- [ ] Notes that discovery motions in particular may be decided on the
      briefs unless a hearing is requested

### Composition

- [ ] Caption per the Georgia caption (`ga-statewide-format`) naming the
      **State Court of Cobb County, State of Georgia** and "Civil Action
      File No." — the proposed order drafted as a **separate** document
      (`ga-draft-order`)
- [ ] Self-represented signature block (no Georgia Bar No.) with a
      certificate of service under O.C.G.A. § 9-11-5

## Common failure modes

- Cites FRCP 37 instead of O.C.G.A. § 9-11-37
- Tells the filer to self-calendar a hearing instead of submitting a
  Rule Nisi for the court to set under USCR 6.3/6.4
- Asserts the USCR 6.2 response window from memory instead of the corpus
- Buries the proposed order inside the motion instead of a separate
  document
