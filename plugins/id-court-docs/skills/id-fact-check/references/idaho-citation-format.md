# Idaho Citation Format

> **NOT LEGAL ADVICE.** Citation-form reference. Verify each cite
> against the source and confirm current usage via `id-law-references`.

Idaho cites to the **official Idaho Reports** together with the
**Pacific Reporter** (West). Idaho has **no neutral / public-domain
(vendor-neutral) citation** â€” do not invent or supply one.

## Cases

Form:

```
[Party A] v. [Party B], [###] Idaho [###], [###] P.[2d/3d] [###] ([YEAR])
```

- A **comma** separates the official Idaho Reports cite and the Pacific
  Reporter cite.
- Keep the periods in `Idaho`, `P.2d`, `P.3d`, and `v.`
- The first page number is the page on which the opinion begins; a
  second number after it is a pinpoint (jump) cite.

**Idaho Supreme Court** â€” parenthetical is the year only:

```
[Party A] v. [Party B], 113 Idaho 730, 747 P.2d 752 (1987)
```

(Volume / reporter / page shown are an illustrative real Idaho Reports
+ P.2d span; substitute the actual cite for the case you are citing.)

**Idaho Court of Appeals** â€” add a court parenthetical, `Ct. App.`,
before the year:

```
[Party C] v. [Party D], [###] Idaho [###], [###] P.3d [###] (Ct. App. [YEAR])
```

Always verify whether a cited opinion is a Supreme Court or a Court of
Appeals decision and that the parenthetical matches â€” a Court of Appeals
case **without** "(Ct. App. YEAR)" is a format error.

> Use neutral placeholder party names ([Party A] v. [Party B]) unless
> you have confirmed the actual case name and cite. Do not fabricate a
> case name or a reporter cite.

I.R.C.P. 7(b)(3)). Pull current rule text from
