# Oral Argument Outline

> **NOT LEGAL ADVICE.** A structuring aid for argument preparation, not
> a script. Adapt to the motion, the record, and the assigned judge's
> practice.

A workable structure for argument on an Idaho civil motion in the
**District Court** or **Magistrate Division**. Address the bench as
**"Your Honor"**; refer to the opposing party as **"Mr./Ms.
[surname]."**

## Structure

1. **Opening** (~30 sec). "May it please the Court, [Name] for the
   [Plaintiff / Defendant]. This is [movant]'s Motion to [Dismiss /
   Compel / for Summary Judgment]. We respectfully ask the Court to
   [grant / deny] for [N] reasons."

2. **Roadmap** (~30 sec). State the points in order: "First... Second...
   Third..." Give the court a map before the detail.

3. **Strongest point first.** Lead with the most persuasive argument.
   Tie each point to the governing **I.R.C.P.**, **I.R.E.**, or
   controlling Idaho case (Supreme Court or Court of Appeals — note the
   "(Ct. App. YEAR)" parenthetical where it applies), and anchor it to a
   specific place in the record (affidavit ¶, exhibit, admission).

4. **Answer the court's questions directly.** When the judge interrupts,
   answer the question first, then return to the roadmap. Do not duck a
   weakness — address adverse authority candidly and distinguish it.

5. **Reserve rebuttal** if you are the movant; use it only to correct
   misstatements and answer the strongest opposing point.

6. **Closing** (~15 sec). "For these reasons, [movant] respectfully asks
   the Court to [grant / deny]. Thank you, Your Honor."

## Preparation notes

- **Know the standard.** Be ready to state the governing standard (e.g.,
  the I.R.C.P. 56 summary-judgment standard) in one sentence. Pull the
  current text from `../../id-law-references/references/court-rules/IRCP-civil-procedure.md`.
- **Tab the record.** Have the key paragraphs and exhibits flagged so a
  cite can be turned to in seconds.
- **Anticipate three questions** the court is likely to ask and draft a
  one-line answer to each.
- **Watch the clock.** If the court sets a time limit, budget it across
  the points and leave room for rebuttal.
