# E-Filing — iCourt / Odyssey File & Serve (I.R.E.F.S.)

> **NOT LEGAL ADVICE.** Filing-mechanics reference. Confirm whether the
> court is live on iCourt for the case type, the document-type labels,
> the size caps, and the fee with the filing court before submitting.

Idaho's statewide e-filing system is **iCourt E-File**, built on **Tyler
Technologies' Odyssey File & Serve**, governed by the **Idaho Rules for
Electronic Filing and Service (I.R.E.F.S.)**.

## Who must e-file

- **Mandatory for attorneys** — under **I.R.E.F.S. 4(a)**, attorneys
  must e-file.
- **Optional for self-represented individuals** — under **I.R.E.F.S.
  4(b)**, a self-represented party **may** e-file but is not required
  to, and may still file on paper at the clerk's window. **Once a
  self-represented party opts in, the party is bound to e-file for the
  life of the case.**
- **E-filing = consent to e-service** — under **I.R.E.F.S. 17**, a party
  who e-files consents to electronic service of subsequent documents.

Confirm these against the rule text; the I.R.E.F.S. rule numbers are
drift-prone, so verify before relying on a specific subpart.

## Document format

- Submit each component as a **separate text-searchable PDF** under
  **I.R.E.F.S. 6**, within the system's size caps.
- Choose the closest matching **document-type label** in the iCourt menu
  for each component — the menus vary by court and case category.

## Workflow

1. Register / log in to an iCourt File & Serve account.
2. Search for the case by case number or party name (or start a new case
   for an initial filing).
3. Select the court, case category, and the **filing / document type**
   for each component.
4. Upload each component as a separate text-searchable PDF per I.R.E.F.S.
   6 (mind the size caps).
5. Pay any filing fee, or attach the **fee-waiver** application.
6. Submit; the system returns a filing confirmation / file-stamp and
   effects electronic service on registered parties (I.R.E.F.S. 17).

## Guide & File

Odyssey's **"Guide & File"** assisted interview is an optional front end
that helps self-represented filers assemble certain case types. It is a
drafting aid, not a substitute for confirming the packet is complete.

## District Court vs. Magistrate Division

Confirm whether the matter is routed to the **District Court** or the
**Magistrate Division** before assuming the filing path — the case
category and document-type menu differ.

## Paper filing (self-represented)

A self-represented filer may file on **paper** under I.R.E.F.S. 4(b):
bring clean copies to the clerk's window during business hours; the
clerk file-stamps the original and returns a stamped copy. Confirm
whether the court accepts mail or drop-box filings and how many copies
(judge's / conformed copy) the court wants.

## Filing vs. service

E-filing through the portal performs e-service only on **registered**
parties (I.R.E.F.S. 17). A party without an e-filing account must be
served by another allowed **I.R.C.P. 5** method, with a **+3-day mail
add-on** where service is by mail. Filing is not service of process —
the initial summons + complaint go out under **I.R.C.P. 4**.
