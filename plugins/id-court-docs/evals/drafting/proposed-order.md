# id-draft-order — proposed order

## Prompt

Draft a proposed order granting my motion for an Idaho District Court.

## Expected triggers

- `id-draft-order`

## Acceptance criteria

- [ ] Produces a **proposed order** with the I.R.C.P. 2 caption and a
      signature line **for the judge only** — the order makes clear
      only the court signs it
- [ ] Frames findings/relief as proposed for the court's
      consideration, not as a decided outcome
- [ ] Accounts for Idaho's separate-document judgment rule
      (**I.R.C.P. 54(a)**) and entry (**I.R.C.P. 58**) where the
      disposition is a judgment, including a Rule 54(b) certificate
      where fewer than all claims/parties are resolved
- [ ] Leaves a dated signature block for the district judge or
      magistrate judge

## Common failure modes

- Adds a signature line for the moving party as the signer of the order
- Conflates an interlocutory order with a final judgment
