# id-law-references — civil-rule lookup

## Prompt

Where do I find Idaho's summary-judgment rule and its timing, and how
should I cite it?

## Expected triggers

- `id-law-references`

## Acceptance criteria

- [ ] Identifies **I.R.C.P. 56** as the summary-judgment rule and
      points to the court-rules corpus for verbatim text
- [ ] Summarizes the timing track: the motion is served a set number
      of days before the hearing and the answering brief a shorter
      set number — reading the current day-counts from the corpus
      rather than asserting them as memorized constants (the corpus
      reflects ≥28-day motion / ≥14-day answering-brief service before
      the hearing)
- [ ] Notes the motion must be filed a set period before trial
      (read from the corpus)
- [ ] Cites in Idaho form (`I.R.C.P. 56(a)` etc.) and reminds that
      verbatim text should be pulled from the corpus before relying on
      a specific figure

## Common failure modes

- Cites the federal Rule 56 timing instead of Idaho's
- Asserts day-counts from memory without deferring to the corpus
