## How to retrieve verbatim text

1. **Public HTML mirror** — open the canonical NY State
   Senate URL above in a browser.
2. **Open Legislation JSON API** — request the API URL
   above with `?key=<NYSENATE_API_KEY>` appended. The key is
   free to obtain at
   https://legislation.nysenate.gov/static/docs/html/#api-key-registration

   (or the workflow secret of the same name), the next
   quarterly refresh will replace this stub with verbatim
   statutory text.
