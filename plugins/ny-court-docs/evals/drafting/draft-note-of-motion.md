# ny-draft-note — Notice of Motion (CPLR 2214)

## Prompt

I need a Notice of Motion for my NY case. The case is in NY
County Supreme Court, Index 654321/2025, IAS Part 38. I am
moving to dismiss under CPLR 3211(a)(7). I am pro se.

## Expected triggers

- `ny-draft-note`
- `ny-statewide-format`
- `ny-nyco`
- `ny-pro-se`

## Acceptance criteria

### Form (Notice of Motion under CPLR 2214(a))

- [ ] Caption matches the underlying action (two-block
      caption with `-against-` separator)
- [ ] Index No. 654321/2025 + IAS Part 38 displayed
- [ ] Title in ALL CAPS: "NOTICE OF MOTION"
- [ ] "PLEASE TAKE NOTICE" opening
- [ ] Returnable date specified, time and Part location
- [ ] Relief sought stated specifically
- [ ] Grounds for the motion: CPLR 3211(a)(7) — failure to
      state a cause of action
- [ ] Time limit warning to opposing party (CPLR 2214(b) —
      answer at least 2 days before return date; 7 days if
      cross-motion)
- [ ] Service mode and date

### Order to Show Cause alternative

- [ ] If movant requests an OSC instead (e.g., emergency
      stay), draft uses CPLR 2214(d) format with TRO request
      and ex parte signature line

### Service timing

- [ ] Service must be at least **8 days** before return date
      (CPLR 2214(b))
- [ ] Add **5 days** for mail service (CPLR 2103(b)(2)) =
      13 days total

## Common failure modes

- Treats the Notice of Motion like a federal Notice of
  Hearing (the function is different — NY's NoM is a
  jurisdictional service-of-motion document)
- Misses the 8-day / 13-day service math
- Forgets to specify the Part (e.g., "IAS Part 38" — not
  just "the Court")
- Forgets the "PLEASE TAKE NOTICE" leader
