# ny-draft-order — Proposed Order (22 NYCRR § 202.48)

## Prompt

Draft a proposed order granting my motion to dismiss in NY
County Supreme Court, Index 654321/2025, IAS Part 38. I am
pro se.

## Expected triggers

- `ny-draft-order`
- `ny-statewide-format`
- `ny-nyco`

## Acceptance criteria

### Form

- [ ] Caption matches the underlying action
- [ ] Index No. + IAS Part displayed
- [ ] Title in ALL CAPS: "ORDER"
- [ ] WHEREAS clauses summarize relief sought and procedural
      posture
- [ ] ORDERED clauses state the specific relief in plain
      language ("ORDERED that the complaint is dismissed as
      against [defendant] with prejudice")
- [ ] Signature line for the Justice ("J.S.C.") with
      handwritten signature space + line for date and county

### 22 NYCRR § 202.48 settle-order rules

- [ ] If the prompt indicates the order was "directed to be
      settled" by the court, the proposed order is submitted
      within **60 days** of the order/judgment
- [ ] **60-day jurisdictional clock**: failing to settle
      within 60 days means the motion is deemed abandoned
      (the strict-but-lenient interpretation post-*Robert
      Marini*)

### NYSCEF / submission protocol

- [ ] Proposed order labeled clearly so the Part Clerk can
      route it
- [ ] Pro-se signature block omits OCA bar number / NY
      attorney registration

## Common failure modes

- Skips WHEREAS clauses entirely (uncommon in NY proposed
  orders even for short relief)
- Drafts as a federal-style order
- Misses the 60-day settle-order clock at § 202.48
- Drafts the order with operative ORDERED clauses but no
  signature/date line for the Justice
