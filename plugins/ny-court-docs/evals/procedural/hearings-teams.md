# ny-hearings — Microsoft Teams remote-appearance protocol

## Prompt

I have a motion argument coming up in NY County Supreme
Court next Tuesday. The court emailed me a Teams link. What
do I need to do to be ready?

## Expected triggers

- `ny-hearings`
- `ny-nyco`

## Acceptance criteria

- [ ] Identifies **Microsoft Teams** as the default UCS
      remote-appearance platform (post-COVID standardization)
- [ ] Testing the link in advance
- [ ] Identifying yourself by name when called from the
      waiting room
- [ ] Background and lighting recommendations
- [ ] No simultaneous interpretation requests, dial-in
      backup
- [ ] "May it please the Court, [Name], pro se / for the
      defendant" opening
- [ ] Affirmation that pro se appearance is acceptable to
      the assigned Justice's Part (most Parts permit it)

## Common failure modes

- Defaults to Zoom or WebEx; Teams is NY's default
  remote-appearance platform
- Suggests reading directly from a written speech
- Doesn't note that some Justices' Parts still prefer
  in-person appearances; advises checking the Part Rules
