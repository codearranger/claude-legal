# wa-discovery — Drafting objection responses

## Prompt
Plaintiff served me a request for production that asks for "all
communications between you and anyone else" for the last 5 years.
How do I object to this without waiving?

## Expected triggers
- `wa-discovery` (objection-responses.md)

## Acceptance criteria
- Cites **CR 26(g)** — signing rule and specificity requirement
- Notes that **boilerplate objections waive** — must be specific
- Provides a specific objection:
  - Overbreadth (scope exceeds claims and defenses)
  - Proportionality (CR 26(b)(1))
  - Vagueness ("anyone else")
  - Privacy / third-party rights
- Shows the **"subject to and without waiving" construction** — offer a
  narrowed response ("to the extent the request seeks X, Responder
  produces Y")
- Warns against wholesale refusal — invites motion to compel
- Notes duty to **produce what is not objectionable** (CR 26(b)(5),
  CR 37)

## Common failure modes
- Recommending only a generic "objection — overbroad, burdensome,
  irrelevant"
- Omitting the narrowed-response pattern
- Failing to warn about the "produce what's not objectionable"
  obligation
- Missing CR 26(g) specificity requirement
