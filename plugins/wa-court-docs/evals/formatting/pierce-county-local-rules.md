# wa-pierce — Pierce County Superior Court conventions

## Prompt

I have a civil case in Pierce County Superior Court (Tacoma).
What's the local-rule landscape and how do I file?

## Expected triggers

- `wa-pierce`
- `wa-statewide-format`

## Acceptance criteria

- [ ] Identifies court address: 930 Tacoma Avenue South,
      Tacoma 98402
- [ ] References **PCLR (Pierce County Local Rules of
      Superior Court)** overlaying CR + GR
- [ ] PCLR 7 motion-day mechanics: motion 9 court days
      before; response 6 court days before; reply 3 court
      days before
- [ ] PCLR 16 standing case-management order
- [ ] **LINX e-filing system** — Pierce County's electronic
      filing portal
- [ ] **MAR** participation: cases ≤ $100,000 referred to
      mandatory arbitration under RCW 7.06
- [ ] Family-law department + family law commissioners
- [ ] Remote-appearance General Order — WebEx default
- [ ] References Pierce County's appellate context
      (Division II)

## Common failure modes

- Wrong court address
- Using statewide PCLR-equivalent local rules instead of
  PCLR
- Missing MAR participation
- Confusing LINX with statewide eFile or LASC's CRS
