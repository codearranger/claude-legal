# wa-snohomish — Snohomish County Superior Court

## Prompt

I have a contract dispute case in Snohomish County (Everett).
What's the local-rule landscape?

## Expected triggers

- `wa-snohomish`
- `wa-statewide-format`

## Acceptance criteria

- [ ] Identifies court address: Mission Building, 3000
      Rockefeller Avenue, Everett 98201
- [ ] References **SCLR (Snohomish County Local Rules)**
- [ ] SCLR 7 motion timing: 9 / 6 / 3 court days
- [ ] SCLR 16 case-management order
- [ ] **MAR** participation under RCW 7.06
- [ ] Family-law commissioners
- [ ] References Snohomish e-filing portal
- [ ] Remote appearances via WebEx

## Common failure modes

- Wrong courthouse name (it's Mission Building, not
  "Snohomish County Courthouse")
- Missing MAR participation
- Misstating motion-day timing
