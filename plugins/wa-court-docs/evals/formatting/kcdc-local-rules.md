# wa-kcdc — KCDC local rules compliance

## Prompt
I'm filing in KCDC Burien. What local-rule specifics do I need to get
right?

## Expected triggers
- `wa-kcdc`
- `wa-law-references` (local-rules.md)

## Acceptance criteria
- **Clerk-issued date** required on cause number — not just the
  numeric portion; KCDC format is "NNCIVNNNNNNKCX"
- **Working copies**: judge-specific delivery requirements; note the
  Burien civil motion docket routes working copies via
  KCDC.CivilMGT@kingcounty.gov
- **Filing deadlines**: e-filing cutoff times (typically 4:30 p.m.
  local)
- **Remote hearings**: Zoom expectations — camera on, silent unless
  speaking, appear in professional setting
- **Division-specific**: Burien = South Division, Judge Laumann (BU2);
  other divisions have their own judges and courtrooms
- **Motion scheduling**: CRLJ 6(d) timing applies (5 court days for
  motion, 3 for response, noon day before for reply)
- Courtesy copies to opposing counsel / party on same day as filing

## Common failure modes
- Using cause number without clerk date
- Wrong routing address (using judge's personal email instead of
  CivilMGT)
- Missing the 4:30 p.m. filing cutoff
- Using superior-court (KCLCR) local rules for a district court
  matter
- Wrong judge for division
