# Integration — Dissolution packet (filing in King County)

## Prompt

Help me build the filing packet for a King County Superior
Court dissolution. We have two children (ages 5 and 9). I'm
the petitioner. We agree on most things but not parenting
time. Pro se.

## Expected triggers

- `wa-family-court`
- `wa-family-law`
- `wa-kcsc`
- `wa-pro-se`
- `wa-statewide-format`
- `wa-file-packet`

## Acceptance criteria

### Packet contents (using AOC mandatory forms per GR 31)

- [ ] FL Divorce 101 — Petition for Dissolution of Marriage
- [ ] FL All Family 001 — Summons
- [ ] FL All Family 117 — Confidential Information form
- [ ] FL All Family 140 — Proposed Parenting Plan (with
      identified disagreement areas)
- [ ] FL All Family 130 — Proposed Order of Child Support
      + WSCSS Worksheets
- [ ] Financial Declaration
- [ ] FL Divorce 102 — Notice of Confidential and Sealed
      Documents (where applicable)
- [ ] Filing fee + fee waiver (FL All Family 022) if
      indigent

### Across the packet

- [ ] Consistent caption — Superior Court of Washington,
      King County
- [ ] AOC mandatory form versions current
- [ ] Pro Se signature block — name + address + phone +
      email + "Pro Se" / "Self-Represented" designation
- [ ] No PII beyond what GR 22 / GR 31.1 redaction allows
- [ ] References mandatory parenting seminar requirement
- [ ] References 90-day waiting period under RCW 26.09.030

## Common failure modes

- Drafting custom forms instead of using AOC mandatory
- Missing parenting plan or child support worksheets
- Forgetting Confidential Information form
- Missing fee waiver where applicable
