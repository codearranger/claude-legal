# Pro Se Toolkit — Common Motion Types

Quick reference for the most common motions a pro se civil defendant will
file in Washington state court, with drafting notes for each.

## Answer to Complaint

- Must be filed within 20 days of service of the summons (CR 12(a))
- Admit, deny, or state lack of knowledge for each allegation
- Include affirmative defenses (FDCPA statute of limitations, lack of
  standing, failure to state a claim, etc.)
- Include **counterclaims** in the same document if available

## Motion to Dismiss (CR 12(b))

Grounds:
- Lack of subject matter jurisdiction (12(b)(1))
- Lack of personal jurisdiction (12(b)(2))
- Insufficient service of process (12(b)(4) / (5))
- Failure to state a claim (12(b)(6))
- Failure to join a party (12(b)(7))

Must be filed before the answer (or as a combined motion).

## Motion to Compel Discovery (CR 37)

Used when opposing party has failed to produce requested documents or
to respond adequately to interrogatories / requests for admission.

Requirements:
- **Meet and confer** first — required under CR 26(i) / CRLJ 26(f);
  document the attempts
- Identify specifically which responses are deficient
- Request a specific remedy (production, fuller response, deemed
  admission)
- Include a request for **expense-shifting** under CR 37(a)(4)

## Motion for Default Judgment (CR 55)

Used when opposing party fails to answer or respond.

- Requires a Declaration of Service proving the summons was served
- Requires a Declaration of Default showing no response was filed
- Proposed Order of Default must be submitted with the motion
- Damages must be proved by declaration or at a hearing

## Motion to Strike

Used to strike improper filings (late filings, unauthorized replies,
irrelevant material, scandalous content under CR 12(f)).

- Identify the specific portion to strike
- State the rule that makes the content improper
- Request the precise relief (strike, disregard, redact)

## Supplemental Declaration

Used to add evidence after the original declaration was filed — new
facts, corroborating exhibits, updated data.

- Label clearly: "SUPPLEMENTAL DECLARATION OF [NAME] IN SUPPORT OF
  [MOTION NAME]"
- Include the same perjury clause
- Include an exhibit list and cover pages
- File and serve well before the hearing (with enough time for
  opposing counsel to respond)

## FDCPA Counterclaim (for consumer debt cases)

Common claims against debt collectors / debt buyers:

- **§ 1692e** — false or misleading representations (including false
  validation expiration, misrepresenting chain of title)
- **§ 1692f** — unfair practices (attempting to collect amounts not
  owed, filing suit without documentation required by state law)
- **§ 1692g** — failing to provide validation within 30 days of dispute
- **§ 1692k** — damages: actual, statutory ($1,000 per violation, up to
  $1,000 total), attorney fees

Pair with Washington Consumer Protection Act claim (RCW 19.86) via
RCW 19.16.440 per se pathway for enhanced damages.

## CRLJ 11 Sanctions Motion

Held in reserve — deploy only after establishing a pattern of
misconduct through earlier motions.

- Must be filed as a separate motion (not combined with other relief)
- 21-day safe harbor: serve the motion first, wait 21 days before
  filing, allowing opposing counsel to withdraw the offending paper
- Specify the document or conduct sanctioned
- Request the specific sanction (fees, dismissal, other)

## Opposition to Motion

When opposing counsel files a motion, file a response:

- Address each argument in their motion by number or section
- Include a supporting declaration if new facts are needed
- File and serve at least 3 days before the hearing (CRLJ 6(d)(2))
- End with "RELIEF REQUESTED": ask the court to deny the motion

## Reply

When you are the movant and opposing counsel has filed a response:

- File a reply only if new arguments in the response need a response
- Keep it short (≤ 3 pages)
- File and serve **no later than noon the day before the hearing**
  (check current rule)
