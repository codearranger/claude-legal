# Washington State Court Rules

- Source: https://www.courts.wa.gov/court_rules/
- Pulled: 2026-05-02
- Coverage: 1,233 individual rules across 34 rule sets

Verbatim text extracted from the official PDFs published by the Washington
Administrative Office of the Courts. PDF layout extraction is imperfect; cite
the original PDF (linked under each rule) when accuracy matters.

Pull order is preserved within each file: rules appear in the order shown on
the AOC list page (typically rule-number order, with the AOC's section headings).

## How to use

- One Markdown file per rule set (e.g., `CR.md`, `RAP.md`).
- Inside each file, every rule has its own heading, an anchor link, and a "Source" line pointing back to the official PDF.
- Rule body text is fenced as a plain code block to preserve the PDF formatting (paragraph numbering, indented lists, etc.).
- `_manifest.json` is a machine-readable index of every rule set and rule with PDF URLs.

## Re-pulling

To refresh against the live AOC site:

```
python3 scripts/pull_court_rules.py --workers 12 \
  --manifest plugins/wa-court-docs/skills/wa-law-references/references/court-rules/_manifest.json
```

Requirements: `pdftotext` from poppler (`brew install poppler`).
