# Proposed Order Template

A proposed order is the document the judge signs if the motion is granted.
Draft it so the judge only needs to sign — the findings, relief, and
signature block are pre-filled.

## Structure

```
          [COURT NAME, CENTERED, BOLD]
      IN AND FOR THE STATE OF WASHINGTON

[PARTIES]                       │   No. [case number]
                                │
                                │   [PROPOSED] ORDER GRANTING
                                │   [MOTION NAME]

─────────────────────────────────────────────────────────

THIS MATTER came before the Court on [Movant's] [Motion Name]. The
Court, having reviewed the motion, the supporting declaration of
[Name] with exhibits, the response (if any), the reply (if any), and
the records and files herein, and having heard argument of [the
parties / counsel] on [hearing date], finds as follows:

                       FINDINGS

1.  [Finding one — specific fact supporting relief.]
2.  [Finding two.]
3.  [Finding three.]

NOW, THEREFORE, IT IS HEREBY ORDERED:

1.  [Specific relief one — order the respondent to do a specific act.]
2.  [Specific relief two — timing, conditions, consequences.]
3.  [Reservation of rights, if applicable, e.g., "The Court reserves
    jurisdiction to enforce this Order."]

DATED this ___ day of __________, 20__.


                             ______________________________________
                                  JUDGE / COMMISSIONER
                                  [Judge Name, if known]

Presented by:

____________________________________
[Movant Name]
[Role, e.g., "Defendant, pro se"]
[Address]
[Email]

Copy received; [approved as to form / objection reserved]:

____________________________________
[Opposing Counsel Name]
[WSBA #]
[Firm]
```

## Notes

- Label the document "[PROPOSED] ORDER..." — the bracketed word is
  removed when the judge signs and enters the order
- **Findings** should be specific and cited to the record — they
  justify the relief
- **Relief** should be enforceable and clear — "Plaintiff shall produce
  [specific documents] within [X] days" rather than "Plaintiff shall
  comply"
- Include a **signature line for the judge** at the bottom
- Include a **"Presented by"** block with the movant's signature — this
  identifies who submitted the order
- Optionally include a **"Copy received / approved as to form"** line
  for opposing counsel to sign
- Do not pre-fill the date — the judge dates when they sign

## Common types of orders

| Motion | Order title |
|--------|-------------|
| Motion to Compel | Order Granting Motion to Compel Discovery |
| Motion to Dismiss | Order Granting (Denying) Motion to Dismiss |
| Motion for Default | Order of Default / Default Judgment |
| Motion to Strike | Order Granting Motion to Strike |
| Motion for Summary Judgment | Order Granting (Denying) Motion for Summary Judgment |
| Discovery Dispute | Order Resolving Discovery Dispute |
