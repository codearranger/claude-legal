# Declaration Template

A declaration with numbered paragraphs, verification clause, signature
block, and optional exhibit list + cover pages.

## Structure

1. Court header (centered, bold)
2. Two-column caption (parties left, case number + title right)
3. Horizontal rule
4. Salutation: "I, [Name], declare under penalty of perjury..."
5. Numbered paragraphs
6. Verification clause
7. Date and place of execution
8. Signature block
9. Exhibit list (new page)
10. Exhibit cover pages + content

## Scaffolded markdown

```
           [COURT NAME, CENTERED, BOLD]
       IN AND FOR THE STATE OF WASHINGTON

[PARTY A],                      │   No. [case number]
                                │
     Plaintiff,                 │   [DOCUMENT TITLE, ALL CAPS,
                                │    BOLD, WRAPPED]
vs.                             │
                                │
[PARTY B],                      │
                                │
     Defendant.                 │
────────────────────────────────────────────────────────────

I, [Full Name], declare under penalty of perjury under the laws of the
State of Washington as follows:

1.  [First paragraph — identity and purpose. "I am the [role] in the
    above-captioned action. I make this Declaration in support of..."]

2.  **[Bold lead-in for labeled paragraph].** [Body text.]

3.  **[Bold lead-in].** [Body text. Cite exhibits by letter.]

4.  [Continue with numbered paragraphs...]

I declare under penalty of perjury under the laws of the State of
Washington that the foregoing is true and correct.

DATED this ____ day of __________________, 20__.

Executed at [City], Washington.


____________________________________
[Full Name]
[Role designation, e.g., "Defendant, pro se"]
[Address line 1]
[Address line 2]
[Email]
```

## Exhibit list (new page)

```
                    EXHIBIT LIST

Exhibit A:  [Description]

Exhibit B:  [Description]

Exhibit C:  [Description]

Exhibit D:  [Description]
```

## Exhibit cover page (new page per exhibit)

```
                    EXHIBIT A
       [italic caption describing the exhibit]

              [image or content]
```

## Notes

- Paragraph 1 always establishes identity and the purpose of the
  declaration
- Paragraphs with named sub-topics use bold lead-ins so judges can skim
  (e.g., "**Pre-Hearing Baseline.**")
- The verification clause is mandatory and must mirror the salutation
- Include email in the signature block for pro se filers
- Refer to each exhibit by letter in the body at its first mention
  (e.g., "a true and correct copy is attached as **Exhibit A**")
